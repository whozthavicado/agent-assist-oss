# Use Cases

## 1. Sales calls

Help agents respond to objections such as:

- "It's too expensive."
- "I need to think about it."
- "Send me the information."
- "I'm not interested right now."

Possible outputs:
- suggested replies
- short call summary
- next-step recommendation
- CRM note

## 2. Customer support

Help support agents:

- summarize issues faster
- generate cleaner notes
- keep response quality more consistent
- reduce after-call work

## 3. Collections

Help collection agents:

- handle common refusal patterns
- keep notes structured
- identify payment intent signals
- improve follow-up consistency

## 4. Internal QA workflows

Help teams review conversations by generating:

- structured summaries
- quality signals
- follow-up recommendations
- objection-handling visibility
