# Contributing to agent-assist-oss

Thanks for your interest in contributing.

## What this project is about

agent-assist-oss is an open-source framework for building AI tools that help human agents during and after conversations.

The project focuses on practical workflows such as:

- objection detection
- reply suggestions
- conversation summaries
- CRM-ready notes
- QA support

## Ways to contribute

You can contribute by:

- improving documentation
- proposing use cases
- adding example prompts
- improving synthetic datasets
- building backend or frontend components
- reporting bugs or unclear behavior

## Contribution principles

Please keep contributions:

- practical
- easy to understand
- well documented
- aligned with real agent workflows

## Before opening a pull request

Please make sure your contribution:

- has a clear purpose
- matches the scope of the project
- is explained in simple terms
- does not add unnecessary complexity

## Early-stage note

This project is in an early stage, so documentation, architecture, and examples may evolve quickly.
